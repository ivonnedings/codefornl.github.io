Code for NL

Code for NL is het netwerk van open source developers en designers die samenwerken aan de digitale overheid.

Wij helpen overheden

Kennispartner voor beschikbare oplossingen

Op zoek naar bestaande digitale oplossingen voor uw probleem? Of zijn er misschien andere overheden die een oplossing aan het maken zijn? In samenwerking met Waag Society en Europese Projecten als Clarity en OGP Toolbox zorgen we voor een actueel overzicht van projecten en applicaties op het gebied van digitale overheid.

Samenwerkingen tussen overheden

Code for NL stimuleert en faciliteert samenwerkingen tussen overheden bij het ontwikkelen en hergebruiken van software-oplossingen. Ontwikkelaars en ontwerpers kunnen op basis van open source overheden helpen beter samen te werken, bottom-up. Door goed in kaart te brengen welke plannen er zijn en hoe die op elkaar aansluiten, helpen wij overheden elkaar te vinden.

Organiseren van Code Fellowships

Een Code Fellow is een developer of designer, aangesloten op het Code for NL netwerk, die voor een half jaar intern meedraait in uw organisatie. Vanuit die positie helpt de fellow bij het vinden van oplossingen en geeft deze advies over digitale technologie.

E-mail ons voor informatie

Community van developers en designers

Levende standaarden, best practices en bestaande oplossingen

Sluit aan op een netwerk van developers en designers voor uitwisselen van kennis, ervaring en code. In samenwerking met ICTU en Gebruiker Centraal organiseert Code for NL regelmatig Meetups om met elkaar kennis te maken en resultaten te delen.

Internationaal netwerk

Code for NL is aangesloten op internationale netwerken, zoals Code for All en Code for Europe. Jaarlijks zijn er een aantal conferenties waar je vakgenoten kunt ontmoeten van over de hele wereld.

Code Fellowships en vacatures

Code for NL organiseert Code Fellowships bij overheden. Gedurende 6 maanden loopt een Code Fellow intern mee om te adviseren op gebied van digitale mogelijkheden en oplossingen te maken voor de problemen die voorbij komen.
