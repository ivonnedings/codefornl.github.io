Code for NL

Stichting Code for NL is het Nederlandse netwerk van developers en designers die samenwerken aan een open en eerlijke digitale overheid en samenleving.

Ons doel is de succesvolle digitale transformatie van gemeenten, andere overheden en de samenleving als geheel.

Code for NL is governing partner van het internationale Code for All netwerk.

Levende standaarden, best practices, ethische vraagstukken. Sluit aan op een actief netwerk van developers en designers voor uitwisselen van kennis, ervaring en code. In samenwerking met onze partners organiseren we regelmatig meetups om met elkaar kennis te maken, ideeën te delen en samenwerkingen aan te gaan.

Code for NL is aangesloten op het internationale Code for All netwerk van #civictech organisaties. Jaarlijks zijn er een aantal conferenties waar je vakgenoten kunt ontmoeten van over de hele wereld.

Kennispartner voor overheden

Ben je als overheid op zoek naar bestaande oplossingen, informatie over digitale mogelijkheden, of open source projecten van andere overheden om bij aan te sluiten? Wij helpen graag!

In samenwerking met onze partners zorgen we bijvoorbeeld voor een actueel overzicht van projecten en applicaties op het gebied van digitale overheid.

Daarnaast komen we graag langs voor een college of werkgroep over onderwerpen binnen ons vakgebied, zoals open source samenwerking, data science, kunstmatige intelligentie, ethiek en tech, of api's / web services.
