Code for NL

Code for NL is the Dutch network of open source developers and designers working on digital government.

Our goal is the successful digital transformation of municipalities, other governments and society as a whole.

Code for NL is a governing partner of the internationale Code for All network.

Knowledge partner and adviser

Looking for existing solutions, or open source projects to join? Together with Waag Society and the Clarity Project we provide an up-to-date overview of eGovernment projects and applications.

Intergovernmental collaborations

Open source collaborations between governments result in reusable code and solutions, bottom-up. By matching up plans, we help governments find each other.

Fellowships

Add developers and designers to your projects and organisation for fresh new perspectives. This way, solutions can be created and testen fast. Also, our fellows can advise you on technology, data usage and information security.

Hackathons, concept challenges and labs

Sometimes problems are not being solved because good ideas are missing or technological possibilities are unclear. With hackathons, concept challenges and labs you can tap into the creativity of developers and designers to tackle these problems.

E-mail us for an appointment

Developer and designer community

Collaborating and sharing ideas

Living standards, best practices, ethical discussions. Join our network of developers and designers to share knowledge, experience and code. In collaboration with ICTU we organize a Meetup every two months to get to know each other, share ideas and start collaborations.

International network

Code for NL is connected to internationale networks, like Code for All and Code for Europe. Every year we attend conferences where you can meet fellow developers and designers from all over the world.

Fellowships and jobs

Code for NL organises Code Fellowships within government organizations. During a period of 6 months to a year, a Code Fellow is part of the organization to advise on and work on digital technologies and solutions for the problems he or she encounters.
