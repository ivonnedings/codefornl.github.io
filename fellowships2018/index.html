Fellowships 2018

Code for NL

Sinds 2014 organiseren we Code Fellowships bij gemeenten. Ook in 2018 organiseert Code for NL weer een aantal fellowships!

Gedurende een Code for NL fellowship lopen één of meer professionele designers en developers mee bij een (semi-)overheid, om daar ter plekke open source ICT-oplossingen te maken voor de sociale vraagstukken die voorbij komen.

Zoals: Hoe kunnen burgers actueler inzicht krijgen in relevante informatie van hun overheid? Hoe organiseren we meer inspraak op lokaal niveau? En hoe ondersteunen we mensen met een beperking beter in hun gebruik van de openbare ruimte?

Ben je designer of developer en lijkt het je leuk om een half jaar mee te lopen in een gemeente of andere overheid? Of ben je als ambtenaar op zoek naar ICT-professionals die samen met jou een half jaar lang op zoek gaan naar geschikte digitale oplossingen?

Meld je dan nu aan voor een intake-gesprek! Stuur een mail naar info@codefor.nl met daarin je naam, contactgegevens en korte motivatie. Wij nemen dan zo snel mogelijk contact met je op!
