Code for NL zoekt fellows

(English?)

Wil jij de overheid veranderen? En deel uitmaken van een internationaal netwerk van civic innovators?

Code for NL zoekt getalenteerde en ondernemende ontwikkelaars om innovatieve diensten te ontwikkelen. Je werkt in een innovatief internationaal netwerk aan cutting edge applicaties om lokale problemen te tackelen.

Waarom Code for NL?

Code for NL wil, net als haar partners in het internationale Code for All netwerk, creatief ICT-talent injecteren bij de overheid. Om op die manier snel tot gebruiksvriendelijke oplossingen te komen voor maatschappelijke vraagstukken. Fellows voor Code for NL werken vanzelfsprekend altijd open source en met open data: want of het nu gaat om Amsterdam, Helsinki of San Fransisco, veel steden staan voor dezelfde problemen. Het Code for All netwerk zoekt samen naar slimme oplossingen. Een fellow brengt niet alleen technologische kennis in, maar ook een agile manier van denken en werken.

Waarom fellow worden?

  • Als code-fellow werk je direct samen met de beleidsmakers en krijg je zo eerste hand ervaring met de uitdagingen waar gemeentes voor staan.
  • Je draait mee in een landelijk trainingsprogramma waar code-fellows en ambtenaren samen werken aan deze uitdagingen.
  • Als code-fellow word je onderdeel van een internationaal netwerk dat samen de ‘Code for’ waarden actief uitdraagt en kansen voor toekomstige samenwerkingen biedt.
  • Code-fellows krijgen de mogelijkheid internationale trainingen en conferenties te bezoeken.
  • Code fellows worden gecoacht en ondersteund door innovatieve organisaties zoals Waag Society.
  • In Nederland bieden de steden een vergoeding van euro 35.000,- voor deelname aan het fellowship.

Waar moet ik aan voldoen?

  • Je bent een fantastische coder.
  • Je vindt het een uitdaging een op de burger gerichte dienst te ontwikkelen.
  • Je bent sociaal vaardig en wilt partijen verbinden.
  • Je durft tegengas te geven.
  • Je denkt in oplossingen, niet in problemen.
  • Je schema is flexibel genoeg om te reizen wanneer nodig.
  • Je bent bij voorkeur vier dagen per week beschikbaar voor het fellowship.

Waar ga ik aan werken?

Voorbeelden van fellowships die wereldwijd al hebben plaatsgevonden zijn te vinden op www.codeforall.org en www.codeforeurope.net. In Nederland vinden er in 2015 fellowships plaats in Amsterdam en Eindhoven.

Hoe meld ik me aan?

Stuur je CV, motivatiebrief en voorbeelden van relevant werk (portfolio) naar info@codefor.nl
