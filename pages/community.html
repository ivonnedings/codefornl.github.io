Snel groeiend internationaal netwerk

Code for NL is aangesloten op internationale netwerken, zoals Code for All en Code for Europe. Jaarlijks zijn er een aantal conferenties waar je vakgenoten kunt ontmoeten van over de hele wereld. Developers, designers en activisten op het gebied van open overheid, open source, democratie, vrijheid van meningsuiting en vrije journalistiek.

Fellowships

Code for NL organiseert Code Fellowships bij overheden. Gedurende 6 maanden loopt een Code Fellow intern mee om te adviseren op gebied van digitale mogelijkheden en oplossingen te maken voor de problemen die voorbij komen.
